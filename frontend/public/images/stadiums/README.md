# Stadium Images Folder

## 📁 ภาพสนามมวย (Stadium Images)

โฟลเดอร์นี้ใช้สำหรับเก็บภาพสนามมวยทั้ง 4 แห่งที่ใช้ในระบบ

## 📋 รายชื่อไฟล์ที่ต้องมี

คุณต้องวางภาพสนามมวยในโฟลเดอร์นี้โดยใช้ชื่อไฟล์ตามนี้เท่านั้น:

### 1. **rajadamnern.jpg**
   - **สนาม:** สนามมวยราชดำเนิน (Rajadamnern Stadium)
   - **ที่ตั้ง:** กรุงเทพ (Bangkok)
   - **การแข่งขัน:** ทุกวัน (จันทร์ - อาทิตย์)
   - **ขนาดแนะนำ:** 1920x1080px หรือมากกว่า
   - **รูปแบบ:** JPG, JPEG, PNG, WEBP

### 2. **lumpinee.jpg**
   - **สนาม:** สนามมวยลุมพินี (Lumpinee Stadium)
   - **ที่ตั้ง:** กรุงเทพ (Bangkok)
   - **การแข่งขัน:** ทุกวันศุกร์ / เสาร์
   - **ขนาดแนะนำ:** 1920x1080px หรือมากกว่า
   - **รูปแบบ:** JPG, JPEG, PNG, WEBP

### 3. **bangla.jpg**
   - **สนาม:** สนามมวยบังลา (Bangla Boxing Stadium)
   - **ที่ตั้ง:** ภูเก็ต (Phuket)
   - **การแข่งขัน:** ทุกวันพุธ / ศุกร์ / อาทิตย์
   - **ขนาดแนะนำ:** 1920x1080px หรือมากกว่า
   - **รูปแบบ:** JPG, JPEG, PNG, WEBP

### 4. **patong.jpg**
   - **สนาม:** สนามมวยป่าตอง (Patong Stadium)
   - **ที่ตั้ง:** ภูเก็ต (Phuket)
   - **การแข่งขัน:** ทุกวันจันทร์ / พฤหัสบดี / เสาร์
   - **ขนาดแนะนำ:** 1920x1080px หรือมากกว่า
   - **รูปแบบ:** JPG, JPEG, PNG, WEBP

## ⚠️ ข้อสำคัญ

1. **ชื่อไฟล์ต้องตรงกันทุกตัวอักษร** (ตัวพิมพ์เล็กทั้งหมด)
   - ✅ ถูกต้อง: `rajadamnern.jpg`
   - ❌ ผิด: `Rajadamnern.jpg`, `rajadamnern.JPG`, `rajadamnern-stadium.jpg`

2. **ระบบจะล็อคภาพตามชื่อไฟล์** - ถ้าชื่อไฟล์ไม่ตรง ภาพจะไม่แสดง

3. **ถ้ายังไม่มีภาพ** - ระบบจะใช้ภาพ fallback จาก highlights folder

## 📝 วิธีใช้งาน

1. เตรียมภาพสนามมวยทั้ง 4 แห่ง
2. ตั้งชื่อไฟล์ให้ตรงตามรายชื่อด้านบน
3. วางไฟล์ในโฟลเดอร์นี้ (`frontend/public/images/stadiums/`)
4. รีสตาร์ท development server เพื่อให้ระบบโหลดภาพใหม่

## 📍 ตำแหน่งที่ใช้ภาพ

ภาพเหล่านี้จะถูกใช้ใน:
- หน้าเลือกสนามมวย (Booking Section - Step 1)
- การแสดงข้อมูลสนามมวยในระบบ
